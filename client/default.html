Request Header Parser Microservice API

What is this?

This is a microservice API project for Free Code Camp that let you get the IPAdress, Language and Operating System for your browser.

Usage:

https://request-header-parse-ms.herokuapp.com/api/whoami

Example Response:

{"IPAddress":"161.142.90.116","Language":"en-US","Software":"Windows NT 10.0; Win64; x64"}
